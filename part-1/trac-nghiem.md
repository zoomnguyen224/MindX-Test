**## Phần 1: Trắc nghiệm (3 điểm)

Khoanh tròn đáp án đúng nhất.

1. Kết quả của biểu thức print(3 * 'ab') là gì?
   a) 'ab3'
   b) 'ababab'
   c) 'ab ab ab'
   d) Lỗi
   Answer: b) 'ababab'
2. Đoạn mã nào sau đây tạo ra một list chứa các số từ 0 đến 9?
   a) list(range(10))
   b) [0:9]
   c) range(0,9)
   d) list(10)
   Answer: a) list(range(10))
3. Kiểu dữ liệu của x = {1, 2, 3} là gì?
   a) list
   b) dict
   c) set
   d) tuple
   Answer: c) set
4. Câu lệnh if trong Python kết thúc bằng ký tự nào?
   a) {
   b) :
   c) ;
   d) Không ký tự đặc biệt nào
   Answer: b) :
5. Hàm len("Hello") trả về giá trị bao nhiêu?
   a) 4
   b) 5
   c) "Hello"
   d) None
   Answer: b) 5
